Closed Loop Control

In this interface, the system modifies the applied current to try to maintain the desired angle. The control system implemented is a Proportional-Integral-Derivative (PID) controller. The feedback used in this controller is the error, e(t), which is the difference between the desired position (ΘR) and the present measured position (Θmeas(t)) for the current time, t.

The PID controller is then written as follows:

  • Kp: The proportional gain. This gain is multiplied by error to generate the proportional term. The greater the error, the larger the applied current. As you approach the desired position, the current due to the proportional term decreases.
  • Ki: The integral gain. This gain is multiplied by the integral of the error over a time period. This term helps reduce steady state error. If there is a persistent error, this term will increase over time, creating a greater current to be applied.
  • Kd: The derivative gain. This gain is multiplied by the derivative of the error with respect to time. This term can improve response time of the controller as rapid changes in error increase the magnitude of this term. Desired angle (ΘR): The reference angle the arm is trying to reach (in degrees).
