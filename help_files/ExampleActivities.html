For each activity, consider what was challenging and what was easier. In what scenarios might open vs. closed loop controllers be used? How did the different PID gains affect the ability to achieve the task?

  • Activity 1:
    • Select open loop control and start with gravity off.
      • You can set how much current to use and how long to apply it for.
    • Using one motion try to get the angle to 45.
    • Once you are happy with that, try again to get it to 135.
    • Now turn on gravity and try to repeat it.
  • Activity 2:
    • Select closed loop control and start with gravity off.
    • You can modify the proportional term gain and the desired angle to see the effects on system performance. (Set the integral and derivative gains to zero.)
    • Replicate the previous activity of going to 45 and 135.
    • Now turn on gravity and try to repeat it.
    • Repeat this activity with inclusion and modifications to the integral and derivative gains.
